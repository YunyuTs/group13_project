## How to pull request
1.使用git add \<file\> 將要文件加入要更新的版本  
2.使用git commit -m "訊息" 設定這個版本的註解 
3.使用git push 將版本推/送上雲端
```
git add <file>
```
```
git commit -m "輸入你想說的訊息"
```
```
git push
```

## 解決merge conflict
```
git pull --rebase
```
